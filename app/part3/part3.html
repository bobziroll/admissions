A common practice in web development is something called "AJAX." AJAX is basically just a common way to download stuff from the internet.

Oftentimes we've written something incorrectly when we're trying to download some kind of resource from the internet in our application - maybe our CSS doesn't seem to be downloading, or some data we thought we downloaded from an API isn't showing up in our application.

Anytime you're trying to download or upload something from/to the internet (over the NETWORK), it gets logged. If there is an error of some kind, it shows up in RED. You can even preview the error response that the server sent back, which prove very beneficial

We've forced an AJAX call to be made and told the server to fail it. If you find a way to preview the response from the server, you'll find the password to the next section.
