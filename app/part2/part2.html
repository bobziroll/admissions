Part 2

Using CSS to design your site is oftentimes full of "guess-and-check" work.

Fortunately, the developer tools allow us to make changes to an element's CSS right in the browser without having to make any changes to the code itself.

This way, we can test our changes and IMMEDIATELY see what they would look like.

Otherwise, you'd be making changes to the code, switching back to the browser, and refreshing the page EACH TIME you wanted to make a change.

In part 2 of this project, we've hidden an HTML image element from the viewable area of the screen.

In order to move on, figure out how to temporarily change the CSS of that item so that it displays on the screen.
